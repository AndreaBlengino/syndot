gum
===

.. versionadded:: 2.1

``syndot`` automatically detects if gum_ is installed in the system and use it 
for:

- |syndot link|, |syndot unlink| and |syndot diffuse| confirmation to proceed
- |syndot link|, |syndot unlink| and |syndot diffuse| interactive mode to 
  select label(s) to link, unlink or diffuse. It is possible to filter labels 
  and select more than one with :kbd:`Tab` key
